Layout direction

The brain is the hero — which structure gives it the right stage?

brain

A · Immersive

Brain fills the full viewport. A slim floating bar at the bottom holds the text input and run button. Results surface as overlays on the brain. ROI selector accessible via hover or hotkey.

B · Split

Brain occupies ⅔ of the screen. A single right panel holds input, modality selector, and results — scrollable if needed. No left sidebar. Clean two-column layout with clear hierarchy.

C · Command

Thin topbar (brand + 2 icon buttons), brain center, collapsible ROI mini-panel on the right edge, input bar at the bottom. Like a scientific instrument — everything subordinate to the visualization.
