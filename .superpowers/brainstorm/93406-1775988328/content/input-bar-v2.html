A · Segmented control

B · Underline tabs

Tab labels with a blue underline indicator. Lighter weight than segments — feels more like a document editor. Active tab is bold-white, inactive tabs fade to dim.

C · Unified drop bar

A single bar. Drop a file and it attaches above as a chip (showing file type, name, size). Type text and run. Both can coexist — multi-modal input in one pass. No explicit mode switching at all.
