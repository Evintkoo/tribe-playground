3 approaches to improve interactivity

Brain (A) + Stimulus input (C) — pick a scope

A

Focused

Brain: click mesh to toggle region, cursor changes to pointer on hover, hover tooltip shows region name + live activation value

Stimulus: sample text snippets (3–4 quick-fill buttons), Ctrl+V paste image from clipboard, show ⌘↵ hint in text footer

B

Recommended — A + polish

Everything in A, plus:

Brain: hover popover floats near cursor (not just status bar) with region name, BA label, and current σ value. Auto-rotation pauses on any brain interaction (not just drag). Brief flash/ring effect on region click.

Stimulus: 2–3 demo images (gallery of quick-load thumbnails). Video URL auto-loads on Enter key (no separate Load button click).

C

Ambitious — B + extras

Everything in B, plus:

Brain: right-click context menu on region (Select, Zoom to, View info). Paint-select by dragging across brain surface.

Stimulus: browser mic recording for audio input. URL auto-fetch preview for image URLs.

⚠ Significantly more code — higher risk of regressions
